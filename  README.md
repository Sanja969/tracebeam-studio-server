# Tracebeam Studio Server ⚡

Realtime event ingestion server for Tracebeam Studio built with Go, REST APIs, and WebSockets.

---

## Features

- ⚡ Realtime WebSocket event broadcasting
- 📡 REST event ingestion API
- 🧠 In-memory event storage
- 🔄 Live event streaming
- 🚀 Lightweight Go backend
- 🛠 Developer-focused observability tooling

---

## Tech Stack

- Go
- Gorilla Mux
- Gorilla WebSocket
